# Target

## Stack

- Expo SDK 54
- React 19 · React Native 0.81
- TypeScript
- Expo Router 6

## Como rodar

```bash
npm install
npm run start
```

- Android: `npm run android`
- iOS: `npm run ios`

Requer [dev client](https://docs.expo.dev/develop/development-builds/introduction/) (`expo start --dev-client`).
